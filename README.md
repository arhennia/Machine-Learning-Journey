# Machine-Learning-Journey

Welcome to my Machine Learning Journey repository! This repository is dedicated to documenting my progress as I learn Machine Learning from the ground up. Here, you'll find my weekly notes, code implementations, experiments, and mini-projects.

## About The Project

The goal of this repository is to serve as a comprehensive log of my learning process. It is structured week-by-week, making it easy to track the topics covered, ranging from foundational concepts to advanced practical applications.

## Repository Structure

The learning material is organized into weekly folders:

- **[Week-00-Foundation](./Week-00-Foundation/)**: Introduction to AI, ML, and Deep Learning. Covers the basics of supervised/unsupervised/reinforcement learning, the ML workflow, and foundational data structures.
- **[Week-01](./Week-01/)**: *(In Progress)* Exploring further topics and implementations.

Each weekly folder typically contains:
- `README.md`: Overview and summary of the week's topics.
- `Notes.md`: Detailed markdown notes covering concepts and theories.
- `Practice.ipynb`: Jupyter Notebooks with hands-on Python code and exercises.
- `Images/`: Diagrams and screenshots for reference.
- `Resources.md`: Links to helpful articles, tutorials, and external resources.

## Tools & Technologies

- **Language:** Python
- **Environment:** Jupyter Notebooks
- **Core Concepts:** Artificial Intelligence (AI), Machine Learning (ML), Deep Learning (DL)
- *(More tools like Pandas, NumPy, Scikit-Learn, PyTorch/TensorFlow will be added as the journey progresses!)*

## How to Navigate

If you are also learning Machine Learning or looking for a structured way to start, feel free to explore my notes and code!
1. Start from **Week-00-Foundation** to review basic concepts.
2. Read through the `Notes.md` in each week's folder.
3. Open the `.ipynb` files in Jupyter Notebook, VS Code, or Google Colab to run and experiment with the code.
